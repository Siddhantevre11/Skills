# Skills

A curated set of AI-agent workflow skills I use daily, adapted from [mattpocock/skills](https://github.com/mattpocock/skills).

| Order | Skill | Description |
|---|---|---|
| 1 | [grill-with-docs](skills/grill-with-docs/SKILL.md) | A relentless interview to sharpen a plan or design, which also creates docs (ADRs and glossary) as we go. |
| 2 | [domain-modeling](skills/domain-modeling/SKILL.md) | Build and sharpen a project's domain model. |
| 3 | [grill-me](skills/grill-me/SKILL.md) | A relentless interview to sharpen a plan or design. |
| 4 | [to-prd](skills/to-prd/SKILL.md) | Turn the current conversation into a PRD and publish it to the project issue tracker. |
| 5 | [to-issues](skills/to-issues/SKILL.md) | Break a plan, spec, or PRD into independently-grabbable issues using tracer-bullet vertical slices. |
| 6 | [tdd](skills/tdd/SKILL.md) | Test-driven development, red-green loop. |
| 7 | [improve-codebase-architecture](skills/improve-codebase-architecture/SKILL.md) | Scan a codebase for deepening opportunities and present them as a visual HTML report. |

Each `SKILL.md` is copied from the source repo, with a **"How I use this"** section appended describing my specific adaptation.
